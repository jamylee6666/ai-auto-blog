## The Problem: Your Blog Archive Is Gathering Dust

You spent hours writing that comprehensive guide six months ago. It got decent traffic, a few comments, maybe even a social media boost. But now? It's buried in your archive, invisible to new readers and doing zero work for your content strategy.

Here's the frustrating reality: most content creators write a blog post and move on. But smart creators know that *one good blog post can become five, ten, or fifteen pieces of content* across different platforms and formats. A detailed how-to article can become a YouTube script, a LinkedIn carousel, an email course, a social media thread, an infographic outline, and more.

The problem is manually repurposing content is tedious. You'd have to re-read the original, extract key ideas, rewrite for each platform, adjust tone and length—it could take 4–6 hours per piece.

**Enter Claude AI.** With Claude, you can transform a single 2,000-word blog post into multiple repurposed formats in about 30 minutes, with minimal effort and maximum quality. This guide shows you exactly how.

## Step-by-Step: Repurpose Your Blog Post with Claude

### Step 1: Choose Your Source Blog Post and Gather It

Pick a blog post you've written that performed reasonably well or covers an evergreen topic. Length doesn't matter—Claude handles 500-word quick tips and 5,000-word deep-dives equally well.

Open your blog post and **copy the full text** (title, body, everything). Paste it into a Google Doc or keep it ready to copy-paste into Claude.

**Pro tip:** Pick posts that are foundational or solve a specific problem. "The 10 Best Tools for Remote Teams" works better than "My Week in Retrospect."

### Step 2: Log Into Claude and Set Up Your First Prompt

Go to [claude.ai](https://claude.ai) and log in. Create a new conversation.

You're now going to give Claude a "master instruction" that will guide all your repurposing work. Paste this prompt:

```
I'm going to give you a blog post. I want you to help me repurpose it into multiple formats for different platforms. For each format I request, follow these rules:

1. Maintain the core message but adapt the tone and style for the platform
2. Keep the content accurate—don't invent statistics or claims
3. Make it feel natural, not like a robot wrote it
4. Keep it actionable and valuable

Here's my original blog post:

[PASTE YOUR FULL BLOG POST HERE]

Confirm you've read it and are ready to help repurpose it.
```

Send this message. Claude will confirm it's ready to work with your content.

### Step 3: Request Your First Repurposed Format (LinkedIn Post)

LinkedIn is perfect as your first repurposing task because it's moderately constrained but still detailed. Send Claude this prompt:

```
Create a LinkedIn post (150–200 words) based on this blog post. 

Requirements:
- Start with a hook question or insight that stops the scroll
- Extract 2–3 key takeaways
- End with a call-to-action (ask a question to encourage comments)
- Use casual but professional tone
- Include relevant emojis (2–3 max)
- Make it feel like a thought-leadership post, not a promotional link
```

Claude will generate a polished, platform-optimized LinkedIn post in seconds. **You're already saving 15–20 minutes of work.**

### Step 4: Create a Social Media Thread (Twitter/X or LinkedIn)

Now request a threaded format—great for reaching more people on short-form platforms:

```
Create a 5-part Twitter/X thread (each tweet 270 characters max) that breaks down the key insights from the blog post. 

Requirements:
- First tweet is a hook that summarizes the benefit
- Each subsequent tweet adds one key insight or step
- Last tweet includes a CTA (link to full blog post)
- Use simple language—no jargon
- Make each tweet standalone but connected to the thread
```

This format works equally well on LinkedIn as a carousel post. Ask Claude for whichever platform you prioritize.

### Step 5: Generate a YouTube Video Script Outline

If you ever consider video content, a blog post is pure raw material. Request:

```
Create a 10-minute YouTube video script outline based on this blog post.

Requirements:
- Opening hook (20 seconds): grab attention with a question or stat
- Main body: 3–4 key sections, each 2–3 minutes
- Each section should have a clear title and bullet-point breakdown
- Include 2–3 specific examples or demonstrations I could show on screen
- Closing with a CTA (subscribe, comment, visit blog)
- Format as a detailed outline, not a full script
- Tone: conversational, like talking to a friend
```

You now have a roadmap for video without needing to film anything tomorrow. Polish it later or hand it to a video editor.

### Step 6: Create an Email Newsletter Version

If you have an email list, repurposing blog content is gold. Ask Claude:

```
Rewrite this blog post as a newsletter email (300–400 words) for my subscribers.

Requirements:
- Subject line option (try 3 variations)
- Warm opening paragraph (2–3 sentences)
- Condensed main content—hit the top 3 insights only
- Include a "Read the full post" link at the bottom
- Tone: conversational, like an email from a friend
- Add a P.S. line with a secondary CTA
```

Email subscribers are your warmest audience. They appreciate the summary plus the option to dive deeper.

### Step 7: Request an Infographic Text Outline (Optional)

Finally, if you work with a designer or use canva, ask Claude:

```
Create a text outline for an infographic based on this blog post.

Requirements:
- Main headline (benefit-focused)
- 5–7 key stats, facts, or tips (one sentence each)
- Include visual suggestions (icons, colors, layout)
- Make each section easy to understand at a glance
- Include a CTA or social handle at the bottom
```

Hand this to a designer, or use it as a checklist if you're designing it yourself in Canva.

## Advanced Tips: Level Up Your Repurposing

### Tip 1: Build a Content Repurposing Template Library

After repurposing 3–4 blog posts, you'll notice patterns in what works. Create a simple Google Doc with your best prompts for each platform. Next time, copy-paste the exact prompt instead of rewriting it. Claude will produce consistent, high-quality results because the instructions are identical.

### Tip 2: Chain Repurposing Prompts in One Conversation

Don't close Claude after each task. Keep the same conversation open and request all formats in sequence. Claude maintains context and can reference the original post throughout. This is faster and ensures consistency across all versions.

### Tip 3: A/B Test Your Repurposed Content

Before you push all five repurposed versions live, test one on each platform. Monitor engagement rates. Which repurposed format drove the most traffic back to your original blog? Double down on that format for future posts. Claude can adjust the style based on your feedback ("Make it more data-driven" or "Less formal, more funny").
