## 用 ChatGPT 整理客戶名單：Excel 資料清理實戰指南

## 開頭：為什麼你需要用 AI 來清理客戶名單

你是否常常面對這個困擾：客戶資料散落在各個來源，有些重複、有些格式混亂、有些缺少重要欄位？手動逐一修正不僅耗時，還容易出錯。如果你的客戶名單有幾百筆甚至幾千筆，手工整理可能得花上整個下午。**用 ChatGPT 做 Excel 資料清理，不需要懂 VBA 或複雜公式，只要會複製貼上，就能在幾分鐘內完成原本要花幾小時的工作。** 這篇文章會教你如何善用 AI 來自動化這個過程，讓你的客戶名單井然有序。

## 工具準備：你需要什麼

### 必需工具

**1. ChatGPT（免費或付費版本）**
- 免費版本（gpt-3.5）：可應付大部分小型資料清理任務
- 付費版本（gpt-4 或 gpt-4o）：處理更大量資料、更複雜邏輯時精準度更高
- 推薦：先用免費版本試試，如果資料量很大再升級

**2. Microsoft Excel 或 Google Sheets**
- 兩個都可以，Excel 較適合本地操作，Google Sheets 適合團隊協作
- 費用：Excel 通常隨著 Microsoft 365 訂閱（約台幣 165/月），或購買永久版；Google Sheets 免費使用

**3. 文字編輯工具**
- 記事本、Word 或任何文字編輯器都可以

### 時間投入

- 準備資料：5-10 分鐘
- 設計清理指令：5-10 分鐘
- ChatGPT 處理：2-5 分鐘（取決於資料量）
- 驗證結果：5-10 分鐘
- **總計：20-35 分鐘**（相比手工整理可省 80-90% 時間）

## 步驟教學：手把手帶你清理客戶名單

### 步驟 1：準備你的 Excel 檔案並匯出關鍵資料

首先，開啟你的客戶名單 Excel 檔案。假設你有這樣的資料：

| 姓名 | 電話 | 地址 | 備註 |
|------|------|------|------|
| 王小明 | 09123456789 | 台北市信義區 ABC 街 | VIP 客戶 |
| 李美美 | 0912-3456-789 | 新北市板橋區 | - |
| 王小明 | 09123456789 | 台北市信義區 ABC 街 | 重複 |
| 張三 | 電話未提供 | 台中市 | - |

**注意常見的亂象：**
- 重複客戶
- 電話格式不統一（有的有"-"，有的沒有）
- 缺少資料（顯示 "無" 或 "-"）
- 大小寫混淆
- 多餘空格

選取你想清理的資料區域（包括標題列），複製並暫時貼到文字編輯器，這樣更方便貼到 ChatGPT。

### 步驟 2：到 ChatGPT 網站並登入

前往 [www.openai.com](https://www.openai.com)，登入你的帳號（沒有帳號先免費註冊）。開啟新的對話。

### 步驟 3：設計清理指令並貼上資料

這是最關鍵的一步。你的指令要夠詳細，這樣 ChatGPT 才能精準執行。**以下是一個範例指令：**

```
我有一份客戶名單需要清理，請幫我做以下事情：

1. 移除重複客戶（以姓名+電話為判斷依據）
2. 標準化電話格式，全部改成 09XXXXXXXX 或 02XXXXXXXX 的格式（去掉所有"-"和空格）
3. 統一地址格式，去掉多餘空格，縣市+區+街道
4. 移除沒有電話或姓名的行
5. 去掉所有英文和特殊符號，只保留中文數字

以下是我的資料，請清理後回傳給我一份表格（用 markdown 格式）：

[貼上你的 Excel 資料]
```

**提示：** 指令越具體越好。如果你有特殊需求（例如只保留台北市客戶、或把客戶分級），也在指令裡一併說明。

### 步驟 4：複製 ChatGPT 的輸出結果

ChatGPT 處理完成後，它通常會以表格形式回傳清理後的資料。直接複選整個表格，複製到剪貼簿。

範例輸出可能長這樣：

| 姓名 | 電話 | 地址 | 備註 |
|------|------|------|------|
| 王小明 | 0912345678 | 台北市信義區 ABC 街 | VIP 客戶 |
| 李美美 | 0912345678 | 新北市板橋區 | - |
| 張三 | 無電話 | 台中市 | 需聯絡 |

### 步驟 5：貼入 Excel 並驗證

打開 Excel，選擇要貼入的儲存格，**右鍵 → 選擇性貼上 → 貼上值**（如果只貼格式可能會有問題）。

現在檢查一遍結果：
- ✓ 重複客戶有沒有真的被刪除？
- ✓ 電話格式是否統一？
- ✓ 有沒有遺漏重要資料？
- ✓ 特殊情況是否被正確標記？

如果發現問題，回到 ChatGPT 指定修改部分。例如："電話還有一些格式不對，請再檢查一遍，03 開頭的電話應該保持 03XXXXXXXX 的格式"。

### 步驟 6：存檔並建立備份

清理完成後，**立即存檔**，並建議另存一份未修改過的原始版本作為備份。習慣養好了，以後出問題也能回頭參考。

- 減少人為錯誤（AI 不會改錯資料）

- 騰出時間做更有策略性的工作（如客戶分析、跟進策略規劃）

## 進階用法建議

### 進階 1：自動分類和標記客戶

如果你想在清理時順便分類客戶（例如按購買次數、地區、消費金額等級），可以在初始指令加上：

```
除了上述清理工作，請同時：
- 按地區分類（北部、中部、南部、東部）
- 標記是否為新客戶（今年才加入）
- 標記購買頻率（高、中、低）
```

ChatGPT 會根據你提供的資料判斷並自動新增分類欄位。

### 進階 2：定期自動化清理流程

如果客戶名單持續增長，與其每次都手動貼資料，不如：
1. 把清理指令保存起來（建立範本）
2. 每月定期跑一次（例如每月底）
3. 建立一份簡單的清理 Checklist，確保格式始終一致

你也可以搭配 **Zapier** 之類的自動化工具，讓新客戶資料自動流進 ChatGPT 進行初步清理。

### 進階 3：結合 Google Sheets 做團隊協作

如果多人共同維護客戶名單，改用 Google Sheets 並在備註欄記錄：
- 誰進行了清理
- 什麼時間清理的
- 哪些資料是自動補充的

這樣整個團隊都能看到資料源頭，日後有爭議也有跡可循。

## 常見問題速解

**Q：ChatGPT 會不會改錯我的資料？**
A：ChatGPT 通常很可靠，但建議一定要驗證結果。特別是涉及金額、重要識別碼時，檢查一遍是必須的。

**Q：我的資料有幾千筆，ChatGPT 能處理嗎？**
A：可以，但建議分批處理。一次貼 500-1000 筆為佳，避免超出 token 限制。

**Q：免費版 ChatGPT 和付費版差在哪？**
A：付費版（gpt-4）對複雜邏輯的理解更精準，也不受使用次數限制。免費版夠用，但如果經常要清理大量資料，升級會更划算。
